# ZBS

ZBS is a lightweight social network with a custom yellow and black glassmorphism style. The project ships without external dependencies and includes:

- Node.js server
- JSON-based data storage
- Authentication
- Feed, profiles, likes, comments, bookmarks, follows, search and trends
- Yellow and black glassmorphism UI

## Run locally

```bash
node server.js
```

Then open `http://localhost:3000`.
